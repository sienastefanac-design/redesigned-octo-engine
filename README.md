Mental health is becoming a more prominent topic in health care with more research done everyday. Psychologists research panic disorders and want to know what causes them, how people cope with them and which cures/remedies work best. The goal of this project was to take the data set from what the Psychologist researched and turn it into a visual representation using graphs to help people understand visually the impact that mental health has on a group of representatives. 

Using RStudio.
